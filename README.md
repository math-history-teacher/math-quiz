# 数学Ⅱ 確認クイズ + 先生用管理ダッシュボード

高校数学Ⅱの定期試験対策クイズアプリです。  
生徒がクイズに取り組んだ結果を Google スプレッドシートに自動記録し、先生がダッシュボードで学習状況を確認できます。

---

## ファイル構成

```
/
├── index.html        # 生徒用クイズ（出席番号ログイン・結果送信つき）
├── teacher.html      # 先生用ダッシュボード（4タブ）
├── gas/
│   └── Code.gs       # Google Apps Script（バックエンド）
└── README.md         # このファイル
```

---

## セットアップ手順

### ステップ1：Google スプレッドシートを準備する

1. [Google スプレッドシート](https://sheets.google.com) で新規スプレッドシートを作成する
2. シート名（左下のタブ）を **`results`** に変更する

### ステップ2：Google Apps Script にコードを貼る

1. スプレッドシートのメニューから **「拡張機能」→「Apps Script」** を開く
2. 既存のコード（`function myFunction() {}`）をすべて削除する
3. `gas/Code.gs` の内容をコピーして貼り付け、**保存（Ctrl+S）** する

### ステップ3：ウェブアプリとしてデプロイする

1. 右上の **「デプロイ」→「新しいデプロイ」** をクリックする
2. 種類の選択で **「ウェブアプリ」** を選ぶ
3. 設定を以下のようにする：
   - 説明：`数学クイズ用`（任意）
   - 次のユーザーとして実行：**「自分」**
   - アクセスできるユーザー：**「全員」**
4. **「デプロイ」** をクリックする（初回はGoogleアカウントの許可が必要）
5. 表示された **ウェブアプリの URL** をコピーする（`https://script.google.com/macros/s/...` の形式）

### ステップ4：URL を貼り付ける

コピーした URL を以下の2ファイルに貼り付ける。

**`index.html`**（スクリプト先頭付近）：
```javascript
const GAS_URL = 'ここにGASのURLを貼る';
// ↓ このように変更する
const GAS_URL = 'https://script.google.com/macros/s/XXXXXXXXXX/exec';
```

**`teacher.html`**（スクリプト先頭付近）：
```javascript
const GAS_URL = 'ここにGASのURLを貼る';
// ↓ 同じURLを貼る
const GAS_URL = 'https://script.google.com/macros/s/XXXXXXXXXX/exec';
```

> ⚠️ GAS のコードを変更した場合は「新しいデプロイ」で再デプロイしてください。

### ステップ5：GitHub Pages で公開する

1. GitHub でリポジトリを新規作成する（例：`math-quiz`）
2. `index.html`・`teacher.html`・`README.md` をアップロードする（`gas/` フォルダは不要）
3. リポジトリの **「Settings」→「Pages」** を開く
4. Source を **「Deploy from a branch」**、Branch を **`main`**、フォルダを **`/ (root)`** に設定して保存する
5. しばらくすると公開 URL が発行される

### ステップ6：URL を共有する

| 対象 | URL |
|------|-----|
| 生徒 | `https://ユーザー名.github.io/math-quiz/index.html` |
| 先生 | `https://ユーザー名.github.io/math-quiz/teacher.html` |

## 使い方

### 生徒（index.html）

1. 出席番号（1〜40）を入力して「スタート」
2. 名前は省略可
3. 単元を選んで「スタート →」
4. ノートに解いてから「解答を見る」
5. 自己採点（⭕ / ✕）→ 全問終了すると結果が自動送信される

### 先生（teacher.html）

| タブ | 内容 |
|------|------|
| 👥 生徒一覧 | 出席番号・正答率・受験回数。行クリックで推移グラフを表示 |
| 📊 単元別分析 | 単元ごとのクラス平均正答率。単元選択で問題別詳細を表示 |
| 📋 問題別正解率 | 全問題を正解率の低い順にランキング表示 |
| 📈 クラス全体推移 | 日付別のクラス平均推移グラフ。単元フィルターあり |

## 注意事項

- GAS の URL 未設定時でも `teacher.html` はサンプルデータで動作確認できます
- 先生用ダッシュボードは URL を知っていれば誰でも閲覧できます。必要に応じてパスワード管理をご検討ください
- GAS の無料枠（1日あたりの実行回数など）に上限があります。大人数で使用する場合はご確認ください
